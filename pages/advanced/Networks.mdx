# Cybria Networks and Public RPC Endpoints

This reference guide provides a listing of the different Cybria networks and public RPC endpoints.

## Cybria Mainnet

| Parameter                   | Value                                                                                    |
| --------------------------- | ---------------------------------------------------------------------------------------- |
| Network Name                | Cybria Mainnet                                                                           |
| Hex Chain ID                | 0x1a05                                                                                   |
| Chain ID                    | 6661                                                                                     |
| Currency Symbol<sup>1</sup> | ETH                                                                                      |
| Explorer                    | [https://cybascan.io](https://cybascan.io)                                               |
| Public RPC URL              | [https://rpc-mainnet.cybria.io](https://rpc-mainnet.cybria.io)                           |
| Connect Wallet              | [Click here to connect your wallet to Cybria Mainnet](https://chainlist.org/chain/6661)  |
